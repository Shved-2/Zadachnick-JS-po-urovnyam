Задачник JavaScript по уровням Уровень 2.1

Task 01

Дана некоторая строка. Найдите позицию первого нуля в строке.

Task 02

Выведите в консоль все числа в промежутке от 1 до 1000, сумма первой и второй цифры которых равна пяти.

Task 03

Дан массив. Удалите из него элементы с заданным значением. [3, 12, 4, 2, 1, 0, 6, 5, 4, 7, 9, 56, 45, 78, 98, 12, 13, 14, 15, 56, 18, 19];

Task 04

Дан некоторый массив, например, вот такой: [1, 2, 3, 4, 5, 6] Найдите сумму первой половины элементов этого массива.
