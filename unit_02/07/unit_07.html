Задачник JavaScript по уровням Уровень 2.7

Task 01

'a bc def ghij' Переведите в верхний регистр все подстроки, в которых количество букв меньше или равно трем. В нашем случае должно получится следующее: 'A BC DEF ghij'

Task 02

Дан символ. Узнайте, в каком регистре этот символ - в верхнем или нижнем.

Task 03

Дано некоторое число, например, такое: 123789 Удалите из этого числа все нечетные цифры. В нашем случае получится такой результат: 28
