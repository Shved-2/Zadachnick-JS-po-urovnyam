Задачник JavaScript по уровням Уровень 2.2

Task 01

Дан массив с числами. Подсчитайте количество отрицательных чисел в этом массиве. [1, 2, 56, -6, 7, -2, -8, 12, -45, 78, -12, -45, -8, 41, 15, 18, -3]

Task 02

Дан массив с числами. Оставьте в нем только положительные числа. [1, 2, 56, -6, 7, -2, -8, 12, -45, 78, -12, -45, -8, 41, 15, 18, -3]

Task 03

Дана строка. Удалите предпоследний символ из этой строки.

Task 04

Дан некоторый массив, например, вот такой: [1, 2, 3, 4, 5, 6] Поделите сумму первой половины элементов этого массива на сумму второй половины элементов
